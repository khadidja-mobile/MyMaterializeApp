Installer Android Studio + jdk 1.8 + Gradle

    Configure > SDK Manager. SDK Platforms : Android 11.0 et Android 10.0

    SDK Tools : Android SDK build-tools | Android Emulator | Android SDK Platform-Tools | Google Play Licensing Library | Intel x86 Emulator Accelerator

    Configure > AVD Manager : Pixel 4XL > install Q > API Level 29 Android 10.0

    installer et dézipper gradle dans C:\

Variables d'environnement

Ajouter :

  • ANDROID_HOME >

    C:\Users\userName\AppData\Local\Android\Sdk\

  • ANDROID_SDK_ROOT >

    C:\Users\userName\AppData\Local\Android\Sdk\

  • GRADLE_HOME >

    C:\gradle-6.8.3-all\gradle-6.8.3\

  • JAVA_HOME >

    C:\ Program Files\Java\jdk18.0_251\

Ajouter à la variable path :

  • Au début :

    C:\Program Files\Java\jdk18.0_251\bin\

  • Puis :

    C:\Program Files (x86)\Common Files\Oracle\Java\javapath

  • à la fin :

    C:\gradle-6.8.3-all\gradle-6.8.3\bin\

Session_Cordova >> MyFirstCordovaApp

Créer dossier et cd Session_Cordova

Lancer les commandes

  • npm install -g cordova
  • cordova -v
  • cordova create MyFirstCordovaApp
  • cd MyFirstCordovaApp
  • cordova platform add android --save
  • cordova build android
  • cordova emulate android
  • cordova clean android s'il le faut
  • désactiver un anti-virus !!!!

config.xml : fichier de configuration

ou :
cordova run browser --target=firefox
